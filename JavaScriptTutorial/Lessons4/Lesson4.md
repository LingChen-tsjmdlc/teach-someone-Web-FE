# 1. HTML

## 1.6 HTML 常用标签

### (5)列表标签

- `​​<ul>`​​​​ - 无序列表【重要】
- `​​<ol>`​​​​ - 有序列表【重要】
  - type - 编号类型(1, A, a, I, i)
  - start - 起始值
- `​​<li>`​​​​ - 列表项【重要】

### (6)表格标签

- `​​<table>`​​ - 定义表格【重要】
- `​​<caption>`​​ - 表格标题【重要】
- `​​<tr>`​​ - 表格行【重要】
- `​​<th>`​​ - 表头单元格【重要】
- `​​<td>`​​ - 表格数据单元格【重要】
- `​​<thead>`​​ - 表格头部【重要】
- `​​<tbody>`​​ - 表格主体【重要】
- `​​<tfoot>`​​ - 表格页脚
- `​​<colgroup>`​​ - 列组
- `​​<col>`​​ - 列属性

### (7) 表单标签

- `​​<form>`​​ - 表单容器
  - action - 提交地址
  - method - 提交方式(GET/POST)
- `​​<input>`​​ - 输入控件
  - type - 类型(text,password,checkbox,radio,submit,reset,button,file,hidden,email,number 等)
  - name - 字段名
  - value - 默认值
  - placeholder - 提示文本
- `​​<label>`​​ - 表单标签
- `​​<textarea>`​​ - 多行文本输入
- `​​<select>`​​ - 下拉列表
- `​​<option>`​​ - 下拉选项
- `​​<optgroup>`​​ - 选项分组
- `​​<button>`​​ - 按钮
- `​​<fieldset>`​​ - 表单分组
- `​​<legend>`​​ - 分组标题
- `​​<datalist>`​​ - 输入建议列表
- `​​<output>`​​ - 计算结果

### (8)语义化结构标签

- `​​<header>`​​ - 页眉
- `​​<footer>`​​ - 页脚
- `​​<nav>`​​ - 导航链接
- `​​<section>`​​ - 文档中的节
- `​​<article>`​​ - 独立内容
- `​​<aside>`​​ - 侧边栏内容
- `​​<main>`​​ - 文档主要内容
- `​​<figure>`​​ - 自包含内容(如图像)
- `​​<figcaption>`​​ - figure 的标题
- `​​<details>`​​ - 额外细节
- `​​<summary>`​​ - details 的标题
- `​​<mark>`​​ - 高亮文本
- `​​<time>`​​ - 日期/时间
- `​​<progress>`​​ - 进度条
- `​​<meter>`​​ - 度量衡

### (9)多媒体标签

- `​​<audio>`​​ - 音频 【重要】
  - controls - 显示控制条
  - autoplay - 自动播放
  - loop - 循环播放
- `​​<video>`​​ - 视频 【重要】
  - controls, autoplay, loop
  - width, height
  - poster - 封面图
- `​​<source>`​​ - 多媒体源(用于 audio/video)
- `​​<track>`​​ - 字幕/章节
- `​​<embed>`​​ - 嵌入外部内容
- `​​<object>`​​ - 嵌入对象
- `​​<param>`​​ - object 的参数
- `​​<picture>`​​ - 响应式图片容器
- `​​<canvas>`​​ - 图形绘制画布
- `​​<svg>`​​ - SVG 矢量图形
