# 1. HTML
## .1 全称
「HTML」(Hyper Text Markup Language): 超文本标记语言
1. 因为它可以加入图片、声音、动画、多媒体等内容（超越文本限制 ）
2. 不仅如此，它还可以从一个文件跳转到另一个文件，与世界各地主机的文件连接（超级链接文本）
## 1.2 HTML骨架格式
```HTML
<!-- 页面中最大的标签 根标签 -->
<html lang="en">
    <!-- 头部标签 -->
    <head>     
        <!-- 标题标签 -->
        <title></title> 
    </head>
    <!-- 主体 -->
    <body>
    </body>
</html>
```
